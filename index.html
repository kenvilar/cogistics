Tailwind via Play CDN

This page uses Tailwind CSS from the Play CDN (@tailwindcss/browser@4.1.14). The base spacing unit is set to 1px using the new v4 @theme token.

Spacing demo (1px scale)

p-1 → 1px
Content area with header padded by exactly 1px.
p-8 → 8px
Header uses 8px padding (because spacing unit is 1px).
p-[13px] → 13px
Arbitrary value utilities still work for non-scale values.

Notes

  • The Play CDN is great for quick prototypes and static pages; for production apps, consider the CLI for tree-shaking and custom plugins.
  • This repo also includes a basic tailwind.config.js. The CDN doesn’t read that file; it’s provided for future CLI setups.
Made with Tailwind v4.1.14 (Play CDN)
